This is the README for gtkballs.


## Requirements

 - a working X11 setup
 - gtk+ version 2.x

## Compiling

```
$ ./configure
$ make
```

## Installing

```
$ make install
```

## Bugs

If you find a bug, tell me what you did (or fix it and send a
patch).

## Development

You can participate in development of GtkBalls by sending patches or pull requests.
